# Vkusotiiki-bg AI

The project implements the following algorithms in Python 3.5:

 - kNN with kd tree

 - Naive Bayes

 - kMeans

 - TF-IDF


#### Installation details:
 - Install packages from requirements.txt in virtualenv

 - For Ubuntu: install also:

 ```sh
 sudo apt-get install python3-tk
 ```
